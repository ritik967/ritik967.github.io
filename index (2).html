💌📘 HTML Notes

  • 👉HTML stands for HyperText Markup Language
  • 👉It defines the structure of a webpage
  • 👉Common tags: <h1>, <p>, <a>, <div>, <img>
  • 👉Semantic tags: <header>, <nav>, <main>, <footer>

💌🎨 CSS Notes

  • CSS = Cascading Style Sheets
  • 👉Used to style and layout web pages
  • 👉Syntax: selector { property: value; }
  • 👉Selectors: class .class, id #id, element div
  • Box model: margin → border → padding → content
  • 👉Flexbox and Grid for layout

⚙️ 💌JavaScript Notes

  • 👉JavaScript adds interactivity to web pages
  • 👉Can manipulate HTML and CSS (DOM)
  • 👉Variables: let, const, var
  • 👉Functions, Events, Loops, Conditions
  • 👉Async JS: setTimeout, fetch(), async/await
  • 👉ES6+: arrow functions, template literals, destructuring
